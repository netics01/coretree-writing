# AGENTS

Use Coretree writing when the user asks for scan-friendly structured output.

- Prefer multi-level bullets over long paragraphs.
- Keep one idea per bullet.
- Use bullet depth for dependency, not importance.
- Put important decisions, conclusions, causes, and results near the top.
- Promote important details with enough context when they would otherwise be buried.
- Use labels only when they improve scanning.
- Do not force every bullet into `Label: value` form.
- Use `:` for short label-value relationships.
- Use `=>` for cause, change, or consequence.
- Prefer 1-3 bullet levels.
- Use 4 bullet levels only when it improves clarity.
- Avoid 5-level bullet trees.
